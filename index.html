About Me

Engineer. Explorer. Dreamer of autonomous worlds.

Fascinated by how machines can learn, adapt, and move with purpose, I enjoy working at the crossroads of robotics and AI, where imagination meets engineering, and where robots are shaped to navigate complex, dynamic environments.

Autonomous Navigation of a Robot using RL Algorithm

An autonomous robot that learns to navigate efficiently using TD3-based Deep Reinforcement Learning, LiDAR, and ROS 2, capable of real-time path planning in both simulated and real-world environments.

View Project

Dynamic Trajectory Planning of a 7DOF Robotic Arm using Deep Reinforcement Learning

Applied Proximal Policy Optimization (PPO) with PyTorch in a ROS–Gazebo environment for real-time control of a Kinova JACO2 arm. The system features dynamic goal updates, a multi-factor reward function, TensorBoard-based performance tracking, and checkpointed training for reliable trajectory learning.
